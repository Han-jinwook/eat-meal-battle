# 프로젝트 비전 (Vision)

## 1. 목적
- 학생들이 매일 급식을 평가·경쟁하며 식습관을 개선

## 2. 페르소나
- **학생**: 재미있는 도전 요소로 참여를 유도  
- **교사**: 대시보드로 만족도를 실시간 확인  
- **학부모**: 자녀 참여 현황을 모바일로 조회

## 3. 핵심 기능
1.  SNS 로그인(Google·Kakao),  특히 탈퇴기능 확인
1-2.  SNS 가입시 '필수값(이메일/별명/프로필이미지/provider/id)    '추가동의정보  '생년월일'  받1-3.  나이 계산, 학생 유무 저장 (6세~19세),,  -> 학생나이 해당자만 '학교선택 페이지 나옴
       학교/학년/반 등록한 학생들만   급식  평가/댓글/좋아요 가능
1-4.  기타 나머지 일반인중  '학생'이  초대하여  가족/학부모로 연결되면  '학부모/가족'  유형
     ( 자기 자녀/가족인 학생과만  댓글 주고받고 보여짐 )
1-5.  그외 나머지는  일반인/교직원  유형으로   모든 페이지  view만 가능
2.  교육부 API 통해   학교 검색,   (수동으로  학년/반 선택)
3.  교육부 API 통해  DB에 등록된 학교들의  당일 급식 메뉴 호출  (매일 오전 10시 실행)
4.  해당학교 학생 한명이라도 급식 사진을찍으면,  해당 학교 전 학생(가입자)에게  알람가고
    급식메뉴/사진이 뜸 (학교 급식 피드)
5.  여기에  별점 평가와 댓글평가/ 답글달기 /좋아요
6.  GPT API 연결 -  당일 학교 급식 메뉴와  유사도 80%이상일 때  급식 이미지 등록 허용
7.  낮 12:30분까지  사진 업로드 없을 시엔, 새로운 버튼 생성
    [급식 이미지 AI생성하기]  - GPT API
    - 학생중 하나가  버튼 클릭하면   해당 학교  메뉴에 맞춰   급식판 이미지생성하여 보여주고
    - 사진 업로드와 동일한 효과
8.  평가가 끝난학생들에겐  '해당 나이에 맞는'  오늘의  급식 퀴즈 제출 ( GPT 가 생성)
9.   나머지는  위 data들에 맞는  각종  view 페이지들

## 4. KPI
- 참여율 80% 달성  
- 만족도 평균 4.0 이상 유지
